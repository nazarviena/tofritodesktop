# 🥟 Salgados Tô Frito Festas - Magnorio Edition

Bem-vindo ao repositório oficial do site **Salgados Tô Frito Festas**, totalmente reformulado e personalizado por **Magnorio**.

Este projeto foi consolidado em uma **Single Page Application (SPA)** de arquivo único para garantir máxima performance e facilidade de deploy no GitHub Pages.

## 🚀 Como Visualizar o Site

O site está publicado e pode ser acessado através do link abaixo:

👉 [**Clique aqui para acessar o site online**](https://nazarviena.github.io/tofritodesktop/ )

## 🛠️ Tecnologias Utilizadas

*   **React 19** + **TypeScript**
*   **Vite 7** (Build System)
*   **Tailwind CSS** (Estilização)
*   **Lucide React** (Ícones)
*   **Wouter** (Roteamento via Hash para compatibilidade total)

## 📁 Estrutura do Repositório

Para manter a simplicidade solicitada pelo Magnorio, o deploy é feito através de um único arquivo:

*   `index.html`: Contém toda a lógica, estilos e estrutura do site em um único lugar.
